# PNGX Market Cap API

Open source core for tracking Papua New Guinea stock exchange market capitalization.

## Open Source Features

- CSV ingestion from PNGX daily reports
- REST API for company data and market cap history
- PostgreSQL/TimescaleDB schema
- Python and JavaScript SDKs
- Docker deployment
