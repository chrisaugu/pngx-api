# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please **do not** open a public issue. Instead, follow these steps:

1. **Email** the security team at `security@pngx-api.org` with a detailed description of the issue, including steps to reproduce, affected versions, and potential impact.
2. **Provide** any relevant logs, screenshots, or proof‑of‑concept code.
3. **Allow** us reasonable time to investigate and address the issue before any public disclosure.

## Supported Versions

We support the latest stable release of the project. Older versions may receive limited security updates.

## Acknowledgements

Thank you to the community for helping keep this project secure.
