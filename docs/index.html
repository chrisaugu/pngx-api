Welcome to the PNGX-API!

PNGX-API is a RESTful api that pulls data from PNGX.com

This API is part of Crisbot.io, my personal bot which is still under heavy development, but I hope you find it useful for your coding projects.

Use these routes to request data on PNGX listed stocks:

  • https://pngx-api.christianaugustyn.me/api/ - view all stocks ticker currently trading on PNGX
  • https://pngx-api.christianaugustyn.me/api/stocks - view all stocks for the current day
  • https://pngx-api.christianaugustyn.me/api/historicals/:ticker - view all quotes belong to the particular :ticker

No API key is required.
